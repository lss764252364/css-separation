文件夹 · 文件 · 结构
===================

```
root/                 # 工程根目录存放所有工程文件，诸如：package.json, gulpfile.js, .editorconfig 等。
├── [.git]
├── [doc]             # 存放诸如：工程说明、指南、参阅资源等文档。
├── [node_modules]    # Node.js 模块存放处。
├── [dev]             # 预处理目录，存放各类需要按某种规则被处理的图片。
├── [dest]            # 生成目录，存放各类按某种规则被处理后的图片。
├── [gulp]            # 存放自动化任务文件。
```
